About the Project

The Reading Practice Story Generator is a web application designed to help elementary school students practice their reading and spelling skills. It generates custom stories based on user-provided parameters.

How It Works

  1. Users input spelling words, a main character, and a setting.
  2. They also set a humor level for the story.
  3. The backend uses AI to generate a unique story incorporating these elements.
  4. The generated story is then displayed on the frontend for the user to read.
